ebola
=====

Auto-updating statistics on the 2014 ebola outbreak in D3 &amp; call to actions for charity.
See the live site at https://ebolastats.com/

###To setup

```
$ git clone
$ npm install
```

###To build:

```
$ npm run-script build
```

###To run locally:

```
$ node parser.js
hit localhost:3000
```

###To deploy:

```
$ git push heroku master
hit ebola.herokuapp.com
```
